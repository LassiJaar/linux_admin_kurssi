# linux_admin_kurssi
Repository Linux Administration -kurssia varten. (Sinne meni anonyymisyys)

# Kansiot menevät seuraavasti:

## Lemp-app
Täältä löytyy Nginx-etusivu.

## Cron_assignment
Täältä löytyy ohjelmat, jotka hoitavat säädatan hakemisen.

## Streamlit (Linux-koneella myapp)
Täältä löytyy Streamlit-palvelun sivut.

## Mqtt-chat
Täältä löytyy mqtt-chat-palvelun vaatimat tiedostot. Index.html ei löydy täältä, eikä sitä ole jaettu julkisesti.
